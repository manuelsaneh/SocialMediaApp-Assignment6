# Social Media App

Welcome to My Social Media App.

## Usage

### Installation

To install and run this app locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/manuelsaneh/SocialMediaApp-Assignment6.git
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the app:

   ```bash
   npm start
   ```
